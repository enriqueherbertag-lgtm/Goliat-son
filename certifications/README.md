# Certificaciones - Goliat-Son

Esta carpeta contiene los planes y documentos de certificación para vuelos espaciales tripulados.

## Contenido planificado

- `nasa_std_3001.md` — cumplimiento de NASA STD-3001 (requisitos para sistemas tripulados).
- `ecss_requirements.md` — requisitos ECSS (Cooperación Europea para Estándares Espaciales).
- `human_rating.md` — plan de certificación para vuelos tripulados.
- `thermal_vacuum_test.md` — plan de pruebas en vacío térmico.

## Certificaciones aplicables

| Certificación | Organismo | Ámbito |
|---------------|-----------|--------|
| NASA STD-3001 | NASA | Sistemas tripulados |
| ECSS-Q-ST-70 | ECSS | Materiales y procesos espaciales |
| MIL-STD-810H | DoD (EE.UU.) | Pruebas ambientales |
| ISS IDSS | NASA / ESA | Sistema de acople internacional |
