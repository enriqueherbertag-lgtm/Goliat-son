# Software - Goliat-Son

Esta carpeta contiene el firmware y software de control para el aterrizaje autónomo.

## Contenido planificado

- `landing_control/` — algoritmo de guía autónoma con radar altimétrico y sensores láser.
- `engine_control/` — firmware para control de motores principales y thrusters.
- `simulation/` — modelos de simulación de aterrizaje (enlace con `/simulations`).
- `telemetry/` — sistema de telemetría para monitoreo remoto.

## Estado

- Lógica de control definida.
- Código pendiente de implementación.
- Simulaciones de aterrizaje en desarrollo.

## Integración

- Compatible con el sistema de control de Odiseo.
- Puede recibir actualizaciones de trayectoria desde tierra o desde la base marciana.
