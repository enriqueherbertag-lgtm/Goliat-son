# Modelos CAD de Goliat-Son

Esta carpeta contiene los modelos 3D del aterrizador.

## Archivos

| Archivo | Descripción |
|---------|-------------|
| `goliat-son.fcstd` | Modelo completo en FreeCAD |
| `goliat-son.stl` | Exportación STL para visualización rápida |
| `renders/` | Capturas de pantalla del diseño |

## Estructura sugerida del modelo

| Componente | Nivel de detalle inicial |
|------------|--------------------------|
| **Cuerpo principal** | Cilindro de 4.5 m diámetro × 6 m alto |
| **Cabina tripulación** | Hemisferio superior con ventanas |
| **Bodega de carga** | Compuerta lateral o rampa |
| **Motores principales** | 4 toberas en la base |
| **Thrusters** | 8 pequeños alrededor del cuerpo |
| **Tren de aterrizaje** | 4 patas plegables con amortiguadores |
| **Escudo térmico** | Base abatible inferior |
