# Hardware - Goliat-Son

Esta carpeta contiene los esquemas y diseños mecánicos del módulo de aterrizaje.

## Contenido planificado

- `engines_specs.pdf` — especificaciones de los 4 motores principales (120 kN cada uno) y 8 thrusters de control.
- `landing_gear.pdf` — diseño del tren de aterrizaje (4 patas con amortiguadores hidráulicos).
- `heat_shield.pdf` — escudo térmico para entrada atmosférica (soporta hasta 2,500°C).
- `docking_system.pdf` — sistema de acople compatible con IDSS.
- `structure_drawings/` — planos CAD de la estructura principal.
