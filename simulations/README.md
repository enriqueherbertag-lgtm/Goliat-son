# Simulaciones - Goliat-Son

Esta carpeta contiene los archivos de simulación para validar la entrada atmosférica, el descenso y el aterrizaje.

## Contenido planificado

- `atmospheric_entry/` — simulación de frenado aerodinámico (CFD).
- `parachute_deployment/` — simulación de despliegue de paracaídas supersónico.
- `propulsive_landing/` — simulación de aterrizaje propulsivo.
- `six_dof/` — simulación de dinámica de vuelo (6 grados de libertad).
