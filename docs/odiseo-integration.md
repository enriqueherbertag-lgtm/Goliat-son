# Integración con Odiseo

Goliat-Son está diseñado para operar como **Módulo de Carga y Transporte (MCT)** de la nave Odiseo.

## Acople

- **Estándar:** IDSS (International Docking System Standard)
- **Ubicación:** En el núcleo de Odiseo, extremo opuesto a la rueda centrífuga
- **Transferencia:** Tripulación y carga pasan por túnel presurizado

## Operaciones en misión Odiseo

| Fase | Acción |
|------|--------|
| **Tránsito** | Goliat-Son viaja acoplado a Odiseo |
| **Aterrizaje** | Se desacopla en órbita marciana y desciende |
| **Despliegue** | Entrega hábitats, paneles, torres ShieldAir |
| **Reabastecimiento** | Utiliza H₂ + O₂ producidos por ShieldAir para volver a órbita |
| **Reacople** | Se acopla nuevamente a Odiseo para futuras misiones |
